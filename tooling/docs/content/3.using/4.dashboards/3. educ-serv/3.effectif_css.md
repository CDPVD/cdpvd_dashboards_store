title: Effectifs scolaires

::alert{type=warning}
Ce tableau de bord fait le suivi des populations d'élèves dans les écoles du CSS/CS. Il établit le nombre d'élèves par école, par niveau scolaire et montre le total d'élèves par niveau et par population au sein du CSS/CS. Il comporte une vue spécifique pour le primaire et une pour le secondaire.
::

# Déploiement
:badge[tag:effectif_css]{type="success"}
:badge[new in v0.7.0]

## Base de données

La base de données `gpi` doit être liée au projet. Référez-vous à la section [linking a database](/using/configuration/linking) pour plus d'information sur la façon de lier une base de données.

## Comptoir 

Le comptoir suivant doit être activé pour le fonctionnement de ce tableau de bord. Référez-vous à la section [enabling a mart](/using/configuration/enabling) pour plus d'information sur la façon d'activer un comptoir.
* `educ_serv`


#### Spécificité de ce projet Dbt
> Mettez à jour le fichier `cssxx_store/dbt_project.yml` en utilisant l'extrait suivant:


```yaml
models:
  core_dashboards_store: # Enable the models from the core repo
    marts:
      educ_serv:
        +enabled: True                  
    dashboards:                                   
      effectif_css:
        +enabled: True
    interfaces: # The dashboard only needs the GPI database
      gpi:
        +enabled: True
```
