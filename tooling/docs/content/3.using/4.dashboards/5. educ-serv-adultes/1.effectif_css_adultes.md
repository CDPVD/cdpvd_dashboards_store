## Databases

La base de données `Jade_adultes` doit être liée au projet. Veuillez consulter la section [liaison d'une base de données](/using/configuration/databases) pour plus d'informations sur comment lier une base de données.

## Marts 

Le comptoir de données (marts) suivant doit être activé(enabled) pour le bon fonctionnement du tableau de bord. Veuillez consulter la section [activation d'un comptoir de données (marts)](/using/configuration/enabling) pour plus d,information sur comment activer un comptoir.
* `educ_serv_adultes`

## Module réussites 

Dans ce module, l'intégrateur ou l'analyste du CSS a la possibilité de choisir les critères de réussite. Sinon, les critères de réussite par défaut seront activés.

#### Dbt project specification
> Mettez à jour votre fichier `cssxx_store/dbt_project.yml` avec l'extrait suivant

```yaml
# cssXX.data.store/dbt_project.yml
models: 
  store:
    marts:
      educ_serv_adultes:
        +enabled: True 
    dashboards:
      educ_serv_adultes:
        effectif_css_adultes:
          +enabled: true
    interfaces:
      jade_adultes:
        +enabled: true

vars:
  dashboards:
    educ_serv_adultes:
      effectif_css_adultes:
        criteres_reussites: (22, 12, 04, 02)
```
